# What is Evernote keyboard shortcuts
To add keyboard shortcut [Evernote](http://www.evernote.com/) Chrome extension.

## Shortcuts
c: New
e: Edit
<Shift> + <Delete>: Delete
<Shift> + <Ctrl> + s: Save and close
j: New item
k: Old item
x: Selected item
s: Sort
/: Search
<Shift> + :: Search notes
<Shift> + ?: Show Help
